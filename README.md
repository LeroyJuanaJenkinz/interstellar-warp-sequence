# Interstellar Warp Sequence

Audio-reactive real-time WebGL/GLSL warp tunnel animation in a single HTML file.

## Features
- 3-layer interstellar tunnel.
- Plasma tunnel mapped cylindrically so it follows the tunnel walls instead of forming a flat circle.
- Nebula clouds from the Shadertoy cloud source.
- Energy streak flares pulsing with bass.
- Motion blur, bloom, feedback trails, and audio-reactive color grading.

## Run
1. Download `interstellar_warp.html`
2. Open it in a modern browser.
3. Choose a track and press Play.

## Performance
Edit `RENDER_SCALE` near the top of the script. Lower values run faster.
